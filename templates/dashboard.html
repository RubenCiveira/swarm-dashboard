Variables de Plantilla para .env

Puedes usar estas variables que serán reemplazadas automáticamente durante el despliegue:

%DB_HOST% Host de la base de datos
%DB_PORT% Puerto de la base de datos
%DB_NAME% Nombre de la base de datos
%DB_USER% Usuario de la base de datos
%DB_PASS% Contraseña de la base de datos
%DB_TYPE% Tipo de BD (mysql/pgsql)
%APP_NAME% Nombre de la aplicación
%APP_URL% URL de la aplicación
%APP_ENV% Entorno (production)

Ejemplo de uso:

APP_ENV=%APP_ENV% APP_NAME=%APP_NAME% APP_URL=%APP_URL% DB_CONNECTION=%DB_TYPE% DB_HOST=%DB_HOST% DB_PORT=%DB_PORT% DB_DATABASE=%DB_NAME% DB_USERNAME=%DB_USER% DB_PASSWORD=%DB_PASS%

💡 Las variables solo se reemplazan si tienes una base de datos configurada para la aplicación.
